# Logisim 电路设计

本项目包含基于 Logisim 的电路设计文件，主要涵盖了加法器和乘法器的实现。

## 主要内容

- **四位串行加法器**：实现基础的串行加法逻辑。
- **原码一位乘法器**：实现原码一位乘法的硬件逻辑。

## 项目来源

本项目相关电路逻辑修改自 GitCode 用户 [bond007](https://gitcode.net/bond007) 的开源项目。

## 使用说明

1. 下载并安装 [Logisim](http://www.cburch.com/logisim/)（建议使用版本 2.15.0.2 或兼容版本）。
2. 打开 `6.原码一位乘.circ` 文件即可查看和仿真电路。
